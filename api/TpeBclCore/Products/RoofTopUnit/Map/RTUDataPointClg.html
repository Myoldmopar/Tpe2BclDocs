Creates a new heat pump water heater data point given specific data point data

Parameters:

  • air_temp (FixNum) —

    Independent variable: Evaporator entering air dry-bulb temperature, degrees Celsius

  • flow_frac (FixNum) —

    Independent variable: Evaporator entering air flow as a fraction of design flow rate, unitless

  • capacity_fraction (FixNum) —

    Dependent variable: Evaporator capacity as a fraction of design capacity, unitless

  • power_in_fraction (FixNum) —

    Dependent variable: Unit input power used as a fraction of design input power, unitless

  • heating_cop_frac (FixNum) —

    Dependent variable: Heating coefficient of performance as a fraction of design coefficient of performance, unitless
