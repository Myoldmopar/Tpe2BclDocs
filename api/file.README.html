tpe-to-bcl

Contains scripts and supporting information for translating TPE products into a form fit for BCL

Call the program with two required (plus one optional) command line arguments which are:

  • A path to a directory containing performance data to process,

  • The product-category key, and

  • An optional flag to define whether to skip the unzipping process for product data (useful during debugging)

TpeBclCore.rb "test-products/HPWH-TestBrandModel" "htpmpw"

Or call the main function from within ruby, passing the directoy as an argument, where it should be relative to the current working directory:

require 'TpeBclCore'
process_dir = "test-products/HPWH-TestBrandModel"
product = "htpmpw"
perf_data, file_paths, comments, go_no_go = TpeBclCore.main(process_dir, product)
